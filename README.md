# rebothead-tracking
基于rk3576开发板和高擎5047_36电机的yolov8辅助机器人头部跟踪项目
  1. 项目概述 - RK3576 开发板 + 高擎5047_36电机的YOLOv8人体跟踪系统    
  2. 架构说明：
    - person_tracking.py -
  主跟踪程序（YOLOv8检测、MJPEG推流、坐标输出）
    - test_motor_rk3576_native.py - 电机控制测试
  3. 串口协议 - UART通信数据包格式（帧头、长度、命令、数据、校验和）   
  4. 常用命令：
    - 运行跟踪程序：python3 person_tracking.py
    - 测试电机：python3 test_motor_rk3576_native.py
    - 查看视频流：http://<RK3576_IP>:8080/stream
  5. 关键实现细节：
    - 坐标输出格式（相对坐标 -1 到 1，适合电机控制）
    - 跟踪算法（IOU匹配 + 5帧移动平均平滑）
    - 电机控制说明（电机21偏航、电机22俯仰）

  1. person_tracking.py
     ├─ 打开摄像头 /dev/video36
     ├─ YOLOv8检测（8fps）
     └─ 输出坐标 → tracker_coords.json

  2. horizontal_tracking.py
     ├─ 读取 tracker_coords.json
     ├─ 计算目标位置（左/右/中）
     └─ 计算增量 → 调用 motor_controller

  3. motor_controller.py
     ├─ 接收增量值
     ├─ 打包数据包（命令0x11）
     └─ 发送到串口 /dev/ttyS8

  4. STM32
     └─ 控制电机21（偏航轴）转动

person_tracking.py 文件讲解
这是一个为 RK3576 开发板设计的 YOLOv8 人体检测 + 跟踪 + MJPEG 推流 综合程序。主要用于摄像头人体跟踪，并将坐标数据输出给电机控制程序。
